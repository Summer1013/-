# 2 IDEA默认快捷键

## 2.1 代码格式化

ctrl + alt + L

## 2.2 复制该行到下一行

也可以是多行,只要选中就行

ctrl + D

## 2.3 调出构造器 异常等

shift + alt + insert(0)

## 2.4 删除该行

ctrl + Y

## 2.5 参数提醒

ctrl + P

## 2.6 搜索

ctrl + F  

先复制再使用快捷键搜索会自动给你开始搜索
